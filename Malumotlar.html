📚 Batafsil Ma'lumotlar

🟦 HTML (HyperText Markup Language)

HTML - barcha veb-sahifalarning skeleti. U kontentni tuzish, ma'noni belgilash va brauzerga qanday ko'rsatishni aytib beradi.

  • Veb-sahifa tuzilishi (DOCTYPE, html, head, body)
  • Matn, rasm, havola, jadval va forma teglari
  • Semantik yozish va SEO optimizatsiya
  • Accessibility (a11y) tamoyillari
  • HTML5 yangiliklari (audio, video, canvas, API)

🟩 CSS (Cascading Style Sheets)

CSS - veb-sahifaning ko'rinishi, dizayni va interaktivligi uchun javob beradi. Ranglar, joylashuv, animatsiya va moslashuvchanlik.

  • Selectors, specificity va cascade qoidalari
  • Box model, margin, padding, border
  • Flexbox va CSS Grid layout tizimlari
  • Responsive dizayn va Media Queries
  • Animatsiyalar, transitions va CSS variables
  • Modern CSS (nesting, container queries, layers)
